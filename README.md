# 🚀 Kubernetes Learning Repository

This repository is used to document and practice my **Kubernetes (K8s)** learning journey.

It contains hands-on examples, experiments, and mini-projects covering core Kubernetes concepts such as:

* Deployments
* Services (ClusterIP, NodePort)
* ConfigMaps
* Secrets
* Pod communication
* Networking & debugging



## 📌 Purpose

The goal of this repository is to:

* Learn Kubernetes through practical implementation
* Build a strong foundation in container orchestration
* Experiment with real-world DevOps scenarios
* Track progress over time


## 📂 Structure

The repository will grow over time and include:

* Sample deployments (MongoDB, apps, etc.)
* Configuration files (YAML manifests)
* Multi-service -setups


## ⚙️ Tools Used

* Kubernetes (Minikube)
* Docker
* kubectl


## 🧠 Note

This is a **learning-focused repository**, so it may contain incremental changes, experiments, and evolving configurations.
